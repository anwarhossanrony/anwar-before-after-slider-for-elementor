Create responsive before and after image comparison sliders in Elementor with smooth drag and touch support.

== Description ==

Anwar Before After Slider for Elementor helps you create clean before and after image comparison sliders inside Elementor.

Useful for:
- Cleaning before and after results
- Renovation and remodeling projects
- Beauty and skincare transformations
- Fitness progress comparisons
- Landscaping work
- Photography editing comparisons

Features:
- Elementor widget for before and after image comparison
- Smooth drag interaction
- Touch support for mobile devices
- Responsive default heights
- Custom before and after labels
- Start position control
- Image fit and image position controls
- Divider and handle styling
- Lightweight frontend assets

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/anwar-before-after-slider-for-elementor` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the Plugins screen in WordPress.
3. Make sure Elementor is installed and activated.
4. Open Elementor and search for "Before / After Slider".
